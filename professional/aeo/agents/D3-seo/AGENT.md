# D3: SEO Agent

## Identity
- **Phase:** D — Quality & Optimization
- **Stage:** Stage 4 (parallel with D1)
- **Purpose:** Traditional on-page SEO optimization.

## Inputs
- Composed draft from C5
- Target URL from content brief

## Process

### Step 1: Title & Meta
- Title tag: includes primary keyword, <60 characters
- Meta description: includes primary keyword, compelling, <155 characters
- URL slug: short, keyword-rich, matches /learn/ structure

### Step 2: Heading Hierarchy
- H1: one per page, contains primary keyword
- H2s: contain related keywords, phrased as questions
- H3s: support H2s logically
- No heading level skips (H1 → H3 without H2)

### Step 3: Keyword Placement
- Primary keyword in: title, H1, first paragraph, URL, meta description
- Secondary keywords distributed naturally in H2s and body
- No keyword stuffing (-8% visibility per GEO research)

### Step 4: Technical Elements
- Image alt text (if images present)
- Internal link anchor text uses descriptive keywords
- External links use rel="noopener" where appropriate
- Reading level appropriate for educator audience

## Output
**SEO Optimization Checklist:**
- Title tag suggestion
- Meta description suggestion
- Heading hierarchy assessment
- Keyword placement audit
- Technical element check
- Suggested edits (specific, not vague)

## Constraints
- SEO optimization must not compromise AEO extractability
- Never stuff keywords — quality over density
- Suggestions should be specific and implementable

## Dependencies
- **Upstream:** C5 (composed draft)
- **Downstream:** Informs final optimization before E1/E2
- **Parallel:** Runs alongside D1, D2, D4
