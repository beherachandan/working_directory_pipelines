## Query Pattern Insights (from calibration)

### Queries that trigger AI web search reliably (ChatGPT)
- Definition queries: "what is X" — always triggers web search
- How-to queries: "how to X in the classroom" — triggers web search
- Comparison queries: "X vs Y" — triggers web search
- Strategy queries: "X strategies for teachers" — triggers web search

### Queries that can return training-data answers (no citation)
- Very generic queries: "what is learning" — GPT answers from training data
- Ambiguous queries: "student engagement" alone (no verb/context)
- **Fix:** Append intent signal — "what is student engagement in K-12 classrooms" > "student engagement"

### Variation design rule (for calibration)
All variations of a topic must be answerable by ONE article (same scope, different phrasing).
If variations would require different articles → they're different topics, run separate calibration.

## High-AEO-Opportunity Query Signals

From keyword research (22 open opportunities) and calibration findings:
- Topics where field avg D2 < 6.0 = high content gap = high AEO opportunity
- Topics where stats field avg < 3.0 = Wayground can dominate with minimal effort
- Topics with multiple GEO gold URLs = competitive but tractable (citation landscape exists)
- Topics with 0–1 GEO gold URLs = either weak AI intent or very authoritative single source
