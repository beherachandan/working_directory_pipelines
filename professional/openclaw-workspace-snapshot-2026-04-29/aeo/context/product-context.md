# Wayground — Product Context for Content Pipeline
_Source: Wayground shared context (April 2026) — canonical product reference for all pipeline agents._
_Imported: 2026-04-28_

## Company
- **Name:** Wayground (formerly Quizizz — rebranded Oct 2025–Feb 2026)
- **Safe external reference:** "Formerly known as Quizizz" on first mention only, through end of 2026
- **Product:** World's most versatile supplemental learning platform (NOT core curriculum / NOT HQIM)
- **200M+ resources** on platform
- **Positioning:** AI-enabled supplemental curriculum hub bridging district priorities and classroom realities
- **Users:** US K-12 teachers, students, school admins, district buyers
- **Revenue:** Freemium — Starter (free), Individual, School, District

## Critical Positioning Rules
- Wayground is **supplemental** — it SUPPORTS implementation of HQIM (Eureka, Bridges, IM, CKLA, Amplify), never positioned as a replacement or as core curriculum
- Wayground is an **instructional hub** — instruction + practice + assessment in one place
- "AI-supported, teacher-first"

## Product Pillars
- **Teacher-facing:** Resources | Joy | Differentiation
- **Admin-facing:** Flexibility | Differentiation | Alignment

## Content Formats (Activity Types)
1. **Assessment / Quiz** — 20+ question types, auto-grading, live or async
2. **Lesson (Presentation)** — slide-based with embedded questions
3. **Interactive Video** — YouTube or uploaded videos with timestamp questions
4. **Passage** — digital reading handouts with inline questions (NOT "reading handout")
5. **Flashcard** — self-study mode

**Always use "activity" as the safe default, never "quiz tool" or "quiz app."**

## AI Features (Wayground AI — Three Capabilities: Create, Enhance, Analyze)
- **Create with AI** — generate activities from prompt, document, or URL
- **AI Teaching Assistant** — on-platform creation + instructional support
- **AI-Powered Answer Explanations** — step-by-step feedback on incorrect answers
- **Differentiated Versions** — easy/medium/hard in a few clicks
- **AI Reading Level Adjustment** — per student or group
- **AI Translation Support** — 18+ languages (Spanish, Vietnamese, Russian, Hindi, Arabic, Chinese, Portuguese, Haitian Creole, Urdu, Hmong, Ukrainian, Romanian, German, Gujarati, Armenian, Hebrew, Tagalog, Yoruba)
- **Analyze with AI** — insights from student responses

## Accommodations
- **25+ accommodations** permanently free for all U.S. educators (as of March 2026)
- Key: Extended Time, Read Aloud, Reduced Answer Choices, Dyslexia Font, Translation (41 languages), Focus Mode, Desmos Calculator, Central Accommodations

## Integrations
Google Classroom, Canvas (LTI), Schoology, Clever, ClassLink, CSV upload

## ICP Profiles

### Teacher ICP
- K-12 classroom teacher, any subject/grade
- Pain: time scarcity, differentiation burden, grading load
- Values: saves prep time, student engagement, easy to use
- Wayground angle: Create activities in seconds, differentiate with one click, AI grader

### Admin ICP
- Principal, curriculum director, district admin
- Pain: alignment to standards, visibility into learning gaps, ROI on tools
- Values: data visibility, standards alignment, teacher adoption
- Wayground angle: Analyze with AI, district-wide reporting, LMS integrations

### Student ICP
- K-12 student
- Pain: boring practice, no feedback, doesn't know what to study
- Values: engaging, game-like, immediate feedback
- Wayground angle: Interactive activities, flashcards, AI explanations
