# SnD Value / Discovery Questions by Feature
*Source: Cody's PMM/Enablement folder • March 2026*

## 1. Question Types
**Value:** Variety aligned to standards (audio/video, hotspot); flexible assessment design for diverse student needs.

**Discovery Questions:**
- What types of assessments do your teachers currently use?
- Are there particular question types your students struggle with or excel at?
- How important is it for your teachers to create custom questions vs. using pre-built content?

## 2. Data Reporting
**Value:** Detailed performance data for teachers and admins; supports monitoring student progress over time and across cohorts.

**Discovery Questions:**
- How does your district currently track student growth and proficiency?
- What types of reports are most valuable to your teachers and administrators?
- Are there gaps in your current reporting that make it hard to make instructional decisions?

## 3. Admin Dashboard
**Value:** Centralized view for administrators to monitor student performance, teacher activity, and assessment results.

**Discovery Questions:**
- How do administrators currently monitor assessment completion and student progress?
- How much visibility do teachers/admins currently have into district-wide performance trends?
- Would a centralized dashboard save time or help drive better decisions?

## 4. Centralized Accommodations
**Value:** Consistent support for students with accommodations across all classes/teachers; reduces teacher burden.

**Discovery Questions:**
- How are accommodations currently managed for students who need them?
- Are there challenges in ensuring accommodations are consistently applied?
- How much time could teachers save if accommodations were automatically applied?

## 5. Common Assessments
**Value:** District-wide alignment with consistent assessment across grades/subjects; enables cross-classroom/school comparison.

**Discovery Questions:**
- Do you currently have district-wide assessments in place?
- How consistent are assessments across classrooms or schools?
- Would having pre-built common assessments help improve instructional alignment?
