## Accommodations (25+)

### Overview
**Teacher:** Set it once, support them always. Support without stigma (personalized versions, same assignment). 25+ accommodations matching real IEPs and 504 plans.
**Admin:** Compliance made simple (every accommodation tracked). Equity at scale. Close achievement gaps with the right support.

### Individual Accommodations
- **Read Aloud:** Text-to-speech for struggling readers, dyslexia, auditory reinforcement
- **Extra Time:** Customized extensions (50%, double, etc.) for processing differences
- **Participant Attempts:** Up to 3 attempts per question — growth mindset in action
- **Translate:** 30+ languages, academic rigor intact
- **Bilingual Dictionary:** Click any word for translation in 18 languages
- **Dyslexia Font:** OpenDyslexic with weighted bottoms, increased spacing
- **Font Size:** Standard to large for visual processing differences
- **Font Spacing:** Extra spacing reduces crowding for dyslexic readers
- **Remove Distractions:** Hide leaderboard for competition anxiety
- **Quiet Mode:** Remove music/sound effects for auditory processing, ADHD, sensory sensitivities
- **Desmos Scientific Calculator:** Built-in for dyscalculia, processing challenges
- **Desmos Graphing Calculator:** Visual math for visual learners
- **AI Hints:** Smart scaffolding — just enough support without giving answers
- **Focus Mode:** Distraction-free, high contrast themes for ADHD, anxiety, visual processing
- **Notes to Refer:** Separate tab with lesson slides during instruction
- **Reading Mode:** Distraction-free, reading-optimized view with alt text
- **Review & Submit:** Second chances for test anxiety
- **Answer Explanations:** AI-powered step-by-step for incorrect answers
- **Reduced Answer Choices:** Fewer options for executive function challenges
- **Question Timer Settings:** Disable/extend timers per IEP requirements

- **Poll:** Real-time class pulse checks, anonymous options
- **Word Cloud:** Collective thinking visualized
- **Audio Response:** Speaking skills, language practice, fluency
- **Video Response:** Performance capture for PE, arts, labs
