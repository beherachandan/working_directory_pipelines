# FAQ Doc: Common Assessments, Objection Handling, Pilot & Pricing
*Source: Cody's PMM/Enablement folder • March 2026*

## What Are Common Assessments?
Wayground Common Assessments bridge classroom realities with district accountability by enabling consistent assessments (unit and benchmark). Key features: Accommodations + lockdown browser + AI-powered insights.

### How It Works
1. **Author:** Create or select assessments; import via Wayground AI
2. **Distribute:** Schedule and assign district-wide
3. **Administer:** Teachers deliver securely to students
4. **Analyze:** AI-powered reports at class, school, and district levels

## Common Objections & Responses

| Objection | Response |
|-----------|----------|
| "We already have district benchmarks" | CA provides easy deployment of those assessments, gathering standardized, actionable data across classrooms and schools |
| "Teachers already have plenty of tools" | CA consolidates high-quality, standards-aligned content with AI insights. Teachers already use Wayground for bell ringers, lessons, exit tickets — now they can deliver unit and benchmark assessments in the same system with test-mode upgrades |
| "Extra work requiring more training" | CA is part of the streamlined Wayground workflow teachers already know, with AI-assisted analysis |
| "We need item banks" | Depends on strategy. Many districts already have high-quality unit tests in their curriculum. CA deploys them consistently across schools with unified data. No need to reinvent the wheel |
| "How is this different from benchmarks?" | Benchmarks: 2-3x/year measuring general growth. CA: unit-by-unit visibility — more frequent, more actionable. District-level data shows which schools need support, where to target interventions. Data arrives in time to reteach |
| "Can't teachers do this in regular Wayground?" | Core Wayground has assessments, but CA adds: secure lockdown mode, synchronized administration, aggregated reporting by subgroups, curriculum pacing verification. Individual classroom data → district-wide insights |

## Pilot Suggestions
- **Start Small:** Begin with a grade level or department
- **Teacher Familiarity:** Practice assessments before official launch
- **Naming Conventions:** Consistent names for organization
- **Performance Bands:** Set mastery/approaching/needs support categories
- **Mini Pilot:** Low-stakes assessment to test workflow and build confidence
- **Duration:** 1-2 units of study; 4-6 weeks suggested
