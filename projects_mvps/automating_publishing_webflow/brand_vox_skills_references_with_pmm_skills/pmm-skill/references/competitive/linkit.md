# Battlecard: LinkIt!
*March 2026 • Confidential • Source: CA Competitive Landscape, Market Survey Data*

## Competitor Snapshot
- **Company:** LinkIt! (private, NJ-based)
- **Products:** Benchmark assessment + predictive analytics, state test correlation
- **Audience:** K-12 districts — strong in NJ/NY (Northeast regional)
- **Market Share:** ~1.2% of district-mandated CFA tools (per CA deck survey)
- **Key Differentiator:** Benchmark assessment with predictive analytics for state test performance; strong state test correlation models for NJ/NY
- **Pricing:** District-level
- **Category:** Tier 2 Assessment-Adjacent Competitor (regional)

## Internal Assessment
LinkIt! is a Northeast regional player with a niche in predictive analytics (predicting state test scores from benchmark data). Strong in NJ and NY but virtually unknown elsewhere. The competitive dynamic is similar to Eduphoria/TX — deeply entrenched in their region, irrelevant nationally.

## Core Weaknesses
- **Regional** — Primarily NJ/NY presence; limited national footprint
- **No engagement** — Traditional assessment interface
- **No daily classroom use** — Benchmark-window only
- **No gamification, AI, or content creation**
- **Traditional assessment feel** — Clinical, test-like experience
- **Limited innovation** — Smaller team, slower feature development

## When You'll See Them
- **NJ and NY districts** primarily — ask about LinkIt! if prospect is in the Northeast
- Strength in districts that value state test prediction models

## Deal Strategy
- **In NJ/NY:** Acknowledge predictive analytics strength. Position Wayground for daily use + engagement. "LinkIt! predicts scores. Wayground improves them — daily engagement drives better outcomes than 3x/year benchmarks."
- **Outside Northeast:** Not a competitive concern.

## THE ONE-LINER
> "LinkIt! predicts test scores. Wayground improves them — with daily engagement that moves the needle all year."
