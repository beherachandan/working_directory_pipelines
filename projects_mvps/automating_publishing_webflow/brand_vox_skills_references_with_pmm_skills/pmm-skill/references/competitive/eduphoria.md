# Battlecard: Eduphoria
*March 2026 • Confidential • Source: CA Competitive Landscape, Market Survey Data*

## Competitor Snapshot
- **Company:** Eduphoria (private, Texas-based)
- **Products:** Assessment (Aware), PD management (Workshop), classroom walkthroughs (Strive), appraisals (Appraise)
- **Audience:** K-12 districts — dominant in Texas, limited national presence
- **Market Share:** ~7.2% of district-mandated CFA tools (per CA deck survey — highest among competitors)
- **Key Differentiator:** Texas-dominant suite covering assessment + PD + walkthroughs + appraisals in one vendor
- **Category:** Tier 2 Assessment-Adjacent Competitor (strong regional, weak national)

## Internal Assessment
Eduphoria is a Texas-specific powerhouse. Their "Aware" assessment module is deeply embedded in Texas districts because it bundles with PD, walkthrough, and appraisal tools. Outside Texas, virtually unknown. The competitive dynamic: in Texas, we're fighting an entrenched vendor with broad school-operations integration. Outside Texas, Eduphoria is irrelevant.

## Core Weaknesses
- **Texas-specific** — Very limited national presence; state test alignment primarily STAAR
- **Aging UX** — Interface feels dated; teachers report it's cumbersome to use
- **No engagement layer** — Pure assessment and admin tool
- **No AI features** — No content creation assistance
- **No gamification** — Students view it as clinical testing
- **Limited outside core subjects** — Focused on core subjects aligned to Texas standards

## Why We Win
1. **National platform vs. regional vendor** — Wayground works in every state with every standard set
2. **Daily classroom use** — Eduphoria is assessment-window only; Wayground is everyday
3. **Teacher experience** — Modern UX with AI creation vs. aging interface
4. **Engagement** — Students engage with Wayground; they endure Eduphoria
5. **Consolidation** — Wayground replaces engagement + assessment tools; Eduphoria is assessment only

## When You'll See Them
- **Almost exclusively Texas districts** — if prospect is in TX, ask about Eduphoria
- Often bundled with their PD/walkthrough tools — "all-in-one operations" pitch
- Strength in districts that want assessment + teacher evaluation from one vendor

## Deal Strategy
- **In Texas:** Acknowledge their operational breadth. Position Wayground as the engagement + daily instruction layer. "Keep Eduphoria for walkthroughs and appraisals. Replace Aware with Wayground for assessment + engagement."
- **Outside Texas:** Not a competitive concern. If mentioned, they're likely a former Texas district admin who brought it along — show broader state alignment.

## THE ONE-LINER
> "Eduphoria manages Texas school operations. Wayground powers daily learning nationwide."
