# Battlecard: Formative (by Newsela)
*March 2026 • Confidential • Source: CA Competitive Landscape, SKO Messaging, Internal Intel*

## Competitor Snapshot
- **Company:** Formative (acquired by Newsela, 2024) — formerly GoFormative
- **HQ:** Los Angeles, CA (now part of Newsela, New York)
- **Founded:** 2013
- **Products:** Real-time formative assessment platform with live monitoring
- **Audience:** K-12 teachers (all subjects) — strength in real-time classroom assessment
- **Market Share:** ~6.0% of district-mandated CFA tools (per CA deck survey data)
- **Key Differentiator:** Real-time teacher monitoring of student work-in-progress; drawing/show-your-work capability; live student response viewing
- **Pricing:** Free tier (limited); paid pricing varies post-Newsela acquisition
- **Category:** Tier 2 Assessment-Adjacent Competitor (formative-focused, not full CA)

## Internal Assessment
Formative is a point solution — real-time formative assessment with a niche in "watch students work live." Post-Newsela acquisition, it's being bundled into the Newsela ecosystem but hasn't gained CA-level capabilities. The competitive dynamic: Formative does one thing well (real-time formative), while Wayground does formative + summative + engagement + CA + accommodations in one platform.

**Important:** Formative (by Newsela) is a different competitive battle than Newsela (content platform). Formative competes in the assessment category; Newsela competes in the content/reading category. Treat them as separate battlecards.

## How They Position
**Messaging pillars:**
- "See student thinking in real-time" — live monitoring of student responses as they work
- "Show your work" — drawing and free-response tools for math/science
- "Formative assessment that actually informs instruction" — real-time data for in-lesson pivots
- "Instant feedback loop" — teachers see understanding immediately

**What they don't talk about:**
- No gamification or engagement layer — students don't love using it
- No common assessment capability — classroom-level only, no cross-school analytics
- No lockdown browser or test security — not built for summative/benchmark use
- No item banks or content library — teachers build everything from scratch
- Limited accommodation suite — not comparable to 25+ accommodations
- No district-level analytics or admin dashboards
- Point solution that doesn't replace other tools (Kahoot, Quizlet, etc.)
- Post-acquisition uncertainty — Newsela's focus is content, not assessment infrastructure

## Why We Win

### Win #1: Platform vs. point solution
> "Formative does real-time formative well. Wayground does that AND summative, CA, gamification, accommodations, and content creation."
Formative handles one slice of the assessment spectrum. Districts still need separate tools for engagement, test prep, common assessments, and content creation. Wayground is the full platform.
- *Landmine:* "Beyond real-time formative checks, what tools handle your common assessments, test prep, and daily engagement?"

### Win #2: Student engagement
> "Assessment data is only as good as student effort. Students engage with Wayground because it feels like a win, not a worksheet."
Formative has no gamification. Student engagement is driven by the teacher monitoring, not the student experience. Wayground's gamification drives completion and effort — which means better data.
- *Landmine:* "Do students voluntarily engage with Formative, or only when directed during class?"

### Win #3: Common Assessment capability
> "Formative is classroom-level. Wayground scales to district-level with CA."
Formative has no cross-school comparison, no district admin dashboards, no coordinated assessment distribution. When districts need common assessment data, Formative can't deliver.
- *Landmine:* "Can your district see how students in Building A performed vs. Building B on the same assessment? Can you aggregate that data across schools?"

### Win #4: Assessment integrity
> "Formative is open-book by design. Wayground offers lockdown browser and test security when you need rigor."
Formative was built for informal, real-time checks — no secure testing mode, no lockdown browser, no anti-cheating controls. When assessments need to be trusted, Formative can't provide that layer.
- *Landmine:* "When you need a secure benchmark assessment, does your formative tool support lockdown mode and anti-cheating?"

2. "Can you see student performance aggregated across schools and grade levels?"
3. "When assessments need test security (lockdown browser, anti-cheating), can Formative provide that?"
4. "What's the total cost of Formative + your engagement tool + your CA tool + your accommodations solution?"
5. "What new features has Formative shipped since the Newsela acquisition?"
6. "How do teachers handle IEP/504 accommodations in Formative? Is it automatic or manual?"

## Deal Strategy

**Scenario A: Formative is entrenched for daily formative checks**
- Don't attack Formative directly — expand the conversation to what it CAN'T do
- "You've got formative covered. What about common assessments, engagement, test prep, and accommodations?"
- Show Wayground handles all of those + formative in one platform

**Scenario B: District evaluating assessment tools**
- Expand evaluation criteria beyond formative: CA, engagement, accommodations, content creation
- "If you're buying a formative tool, Formative works. If you're buying an assessment platform, Wayground does more."

**Scenario C: Newsela bundle offer**
- Separate the content decision from the assessment decision
- "Newsela is content. Formative is assessment. Wayground is both content creation + full assessment spectrum. Compare total value."
