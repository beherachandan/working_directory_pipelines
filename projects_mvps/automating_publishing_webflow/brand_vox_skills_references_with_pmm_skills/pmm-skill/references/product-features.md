## 1. Brand & Product Overview
Wayground (formerly Quizizz) is the world's most versatile supplemental learning platform. Founded in 2015, rebranded Oct 2025 – Feb 2026.

**Terminology:** "Classroom resources," "classroom activities," or "supplemental learning platform" — NOT "quiz tool" or "quiz app." Safe default for external copy: **"activity"**.

**Positioning:** Comprehensive suite for instruction, practice, and assessment. Supports entire instructional cycle: create/discover → instruct → practice → assess → analyze.

| Audience | Tagline |
|---|---|
| Teacher-facing | Let's find a way |
| Admin-facing | Make your curriculum GO |

## 4. Session & Assignment Modes

### Live Session Modes
| Mode | Description | Best For |
|---|---|---|
| **Teacher-Led (Classic)** | Teacher controls pacing; everyone moves together | Whole-class instruction, guided review |
| **Student-Paced (Live)** | Students progress at own pace during live session | In-class practice, formative checks |
| **Test Mode** | Distraction-free, no-frills formal assessment | Summative assessments, benchmarks |
| **Team Mode** | Individual answers, team totals | Collaborative review, gamified competition |
| **Paper Mode** | QR codes for answers; no personal devices required | Low-device classrooms, BYOD alternatives |
| **Focus Mode** | No animations/sounds/clutter; high contrast themes | ADHD, anxiety, sensory sensitivities, visual processing |
| **Mastery Mode** | Practice until mastery; questions adapt to level; no penalty | Homework, skill building, growth mindset |
| **Mastery Peak** | Individualized paths: struggling = foundational; advanced = complex challenges | Differentiated instruction without multiple versions |

| Mode | Description | Best For |
|---|---|---|

| Testing Mode (Async) | Formal; distraction-free; no retries | Take-home tests, remote assessments |

## 10. Common Assessments (CA)
Strategic add-on to Wayground School & District plans. Primary GTM motion for S&D sales.
**Full Help Center reference:** `/home/ec2-user/.openclaw/workspace/references/ca-help-center-reference.md`
**Help Center URL:** https://help.wayground.com/support/solutions/folders/158000652005

**What it is:** District-wide standardized assessments distributed by admins/coordinators to course teachers, administered in a secure testing environment, with multi-level reporting (class → school → district).

**Four-Step Lifecycle:**
1. **Author** — Create standards-aligned assessment (all state-test question types) or import with Wayground AI
2. **Distribute** — Admins/Assessment Coordinators distribute district-wide via Courses (auto-roster from Clever/ClassLink) or Teams (manual groups), set testing window (start/end dates)
3. **Administer** — Teachers run CA in their classes during scheduled window. Students admitted individually. Sessions can be paused/resumed across class periods.
4. **Analyze** — Multi-level reports with performance bands, accountability goals, standards drill-down, subgroup equity analysis, AI-generated insights

**Roles & Permissions (CA-specific):**

| Role | Create CAs | CA Visibility | Report Scope |
|---|---|---|---|
| Admin | ✔️ | All CAs in org | Organization-wide |
| School Admin* | ✔️ | CAs assigned to their school | School-wide |
| Assessment Coordinator* | ✔️ | Created + assigned CAs | Org-wide (created) / Class (assigned) |
| Educator (Teacher) | ❌ | Assigned CAs only | Class-wide |

*School Admin and Assessment Coordinator are specialized roles available only with CA add-on.

**Assessment Integrity:**
- **LockDown Browser** (Respondus) — blocks other apps/websites/browser functions. Supports Mac/Windows/Chromebook. iPads NOT supported. Mass deploy via Google Admin Console.
- **Anti-Cheating Monitor** — full-screen mode with tab-switch notifications (when LDB disabled)
- **Shuffle questions & answer options** — reduces copying risk
- **Student admission gate** — teachers admit each student individually
- **Nightly auto-pause** — prevents unsupervised access outside class hours

**Universal Tools & Accommodations:**
- Per-tool control: Enabled (all students) / Disabled / Teacher Set Accommodation (per-student)
- Calculator, read-aloud, reduce answer choices, etc.
- "Record on Behalf" — teachers enter responses for students needing scribes or who completed offline, with audit trail visible to admins

**Reporting (Admin Report):**
- **Performance Bands**: Did Not Meet / Partially Met / Met / Exceeded (configurable thresholds)
- **Accountability Goals**: Target % of students in Met+Exceeded, spotlighted throughout reports
- **Four analysis tabs**: Standards, Schools, Classes, Sub Groups
- **Deep Dive heatmap**: Group by schools/classes/subgroups/students, absolute accuracy or performance bands
- **Item-level response analysis**: Per-question accuracy, correct answer, option distribution
- **AI Evaluation**: For open-ended questions (teacher reviews AI scores)
- **Downloads**: PDF (branded) and Excel (raw data for custom analysis)
- **Benchmarks**: Class vs. school vs. district at assessment, standard, and item levels

**Teacher Class Report:**
- Participation tracking, class accuracy (best attempts), performance bands
- School & district benchmarks for comparison
- Student status: Pending Evaluation / Not Submitted / Not Started / Scored
- "End Ongoing Attempts" button to finalize after testing window

| Persona | Value |
|---|---|
| Teachers | Administer in secure environment, class-level reports with school/district benchmarks, accommodation support |
| Curriculum/C&I Admins | Consistent district-wide assessment, standards proficiency data, PLC-ready reports |
| Assessment Coordinators | Create & manage CAs, org-wide reporting for their assessments |
| Superintendents/Principals | Evidence of ROI, accountability goal tracking, equity gap analysis via subgroup data |

**Key differentiator:** "Already in the building" — teachers already use Wayground daily for instruction. CA is an upgrade, not a new tool. Only CA platform that combines formative assessment + gamified engagement + summative district-wide testing.

## Paper Mode Clarification
Paper Mode = QR-code-based participation. Students hold up QR codes to answer — NO personal devices required. It is NOT "print, administer, and scan back." Low-device classrooms and BYOD alternatives.
