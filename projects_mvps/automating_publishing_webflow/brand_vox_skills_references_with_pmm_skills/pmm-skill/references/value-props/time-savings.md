# Value Prop: Time Savings

## One-Liner
"Wayground gives teachers hours back every week with auto-grading, a shared content library, and AI-powered quiz generation — so they can focus on teaching, not grading."

## The Data Proof
Query these BigQuery metrics to back this claim:
- **Content library usage** — quizzes used from library vs. created from scratch
- **Auto-graded responses** — total responses auto-scored (manual grading eliminated)
- **AI content generation** — `track.auraGen_page_viewed` for AI quiz creation adoption
- **Quiz reuse** — quizzes used by multiple teachers (shared content efficiency)
- **Teacher activity frequency** — games per week indicates time-efficient workflow

## By Persona

### Superintendent
- Frame: "Support teacher retention by reducing workload — data shows teachers save X hours/week"
- Lead with: Teacher retention, workload reduction, district-wide efficiency
- Proof point: "X teachers are auto-grading Y assessments — that's Z hours of manual grading eliminated"

### Curriculum Director
- Frame: "Shared content library means teachers aren't all building the same assessment independently"
- Lead with: Content sharing across PLCs, consistent quality, AI-assisted creation
- Proof point: "Your teachers have created X quizzes, and Y are shared across buildings — no duplicate work"

### Principal
- Frame: "Teachers who spend less time grading are happier teachers. Happy teachers stay."
- Lead with: Teacher satisfaction, reduced after-hours work, practical time savings
- Proof point: "X teachers running auto-graded assessments weekly — they're getting instant data instead of weekend grading"

### Teacher Champion
- Frame: "Stop grading by hand. Your weekends belong to you."
- Lead with: Auto-grading, AI quiz generation, content library access, import capabilities
- Proof point: "You've auto-graded X responses — premium gives you AI generation and the full content library"

## Competitive Angle
- **vs. Google Forms:** Free but zero auto-grading intelligence, no content library, no AI generation
- **vs. IXL:** Pre-made content only — teachers can't create or customize efficiently
- **vs. Kahoot:** Limited content creation tools, no AI generation, no shared library
- **vs. MagicSchool:** AI-focused but doesn't include delivery + grading + analytics in one platform
- **Wayground advantage:** Create (with AI) → Deliver (as game) → Grade (automatically) → Analyze — all in one workflow
