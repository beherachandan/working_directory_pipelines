# Value Prop: Differentiation

## One-Liner
"Wayground supports every learner with built-in accommodations, accessibility features, and differentiated content delivery — making IEP/504 compliance easier for teachers."

## The Data Proof
Query these BigQuery metrics to back this claim:
- **Accommodation usage** — `analytics_v4.collaboratorTeachersStale` for co-teacher/accommodation assignments
- **Co-teacher assignments** — `clean.co_teacher` for collaborative teaching support
- **Extended time / modified content** usage in game settings
- **Accessibility feature adoption** — language support, text-to-speech usage
- **Diverse content delivery** — multiple game modes serving different learning styles

## By Persona

### Superintendent
- Frame: "Meet your IEP/504 obligations with technology that makes compliance seamless"
- Lead with: District-wide accessibility compliance, equity mandates, special education support
- Proof point: "Accommodations are built into the platform — teachers don't need separate tools for differentiation"

### Curriculum Director
- Frame: "Differentiated instruction that doesn't require teachers to create three versions of everything"
- Lead with: Tiered instruction support, accommodation settings, co-teacher collaboration
- Proof point: "Teachers can set accommodations per student within the same assessment — one quiz, multiple experiences"

### Principal
- Frame: "Your special education teachers and general ed teachers can collaborate seamlessly"
- Lead with: Co-teacher features, building-level accommodation data, ease of IEP compliance
- Proof point: "X teachers using accommodation features — compliance documentation is automatic"

### Teacher Champion
- Frame: "Give every student what they need without creating five different versions of every assignment"
- Lead with: Per-student accommodations, extended time, text-to-speech, modified content
- Proof point: "Set it once per student — accommodations apply to every assessment automatically"

## Competitive Angle
- **vs. Kahoot:** No accommodation features, no per-student differentiation
- **vs. IXL:** Adaptive difficulty but no teacher-controlled accommodations, no IEP support
- **vs. Nearpod:** Limited accommodation features, no co-teacher collaboration
- **vs. Google Forms:** Zero accommodation capabilities
- **Wayground advantage:** Built-in accommodations + co-teacher support + multiple delivery modes = differentiation without extra work
- **Known gap:** Competitors with screen blocking (GoGuardian) have advantage when control is the primary need
