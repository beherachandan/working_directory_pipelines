# Value Prop: Assessment

## One-Liner
"Wayground gives every teacher the power to create, deliver, and analyze formative assessments — and gives districts Common Assessment capabilities to ensure consistency across buildings."

## The Data Proof
Query these BigQuery metrics to back this claim:
- **Quizzes created** — `clean.quiz` count by org/district (teacher content creation)
- **Questions authored** — `clean.question` count (depth of teacher investment)
- **Games run as assessments** — `clean.game` with assessment-mode flags
- **Cross-building consistency** — CA usage across schools in a district
- **Active teachers** — teachers running classroom games (`is_classroom_game = TRUE`)

## By Persona

### Superintendent
- Frame: "District-wide assessment consistency without mandating a single rigid platform"
- Lead with: Cross-building data visibility, standards alignment, teacher autonomy within structure
- Proof point: "X teachers across Y buildings are already creating assessments — premium gives you district-wide visibility"

### Curriculum Director
- Frame: "Common Assessment that teachers actually want to use because they helped build it"
- Lead with: CA capabilities, standards mapping, formative assessment cycles, PLC integration
- Proof point: "Teachers create their own assessments aligned to your standards — CA feature lets you standardize across buildings"

### Principal
- Frame: "See which students get it and which don't — building-wide, in real time"
- Lead with: Building-level assessment data, teacher adoption (low lift), quick implementation
- Proof point: "X teachers already running assessments — premium admin view shows you building-wide trends"

### Teacher Champion
- Frame: "Stop spending your weekends grading — get instant data on who gets it and who doesn't"
- Lead with: Auto-grading, instant feedback, question-level analytics, differentiation data
- Proof point: "You've run X assessments reaching Y students — premium gives you deeper item analysis and reports"

## Competitive Angle
- **vs. Kahoot:** Kahoot is engagement-only — no real assessment analytics, no item-level data, no CA
- **vs. IXL:** IXL is practice, not formative assessment — teachers can't create their own content
- **vs. Nearpod:** Nearpod embeds assessments in lessons but lacks standalone assessment power and CA
- **vs. Google Forms:** Free but zero analytics, no auto-grading, no standards alignment, no CA
- **Wayground advantage:** Teacher-created + library content, real assessment analytics, Common Assessment for districts
