# Board Meeting Keywords → Value Prop Mapping

Use this mapping to connect district board meeting topics (from `wayagent_GTM.account_feature_store`) to Wayground value propositions.

## How to Use
1. Query board meeting notes for the target district
2. Extract keywords/topics from AI summaries
3. Map to value props below
4. Prioritize messaging based on most frequent/recent board topics

## Priority Rules
1. **Most recent board meeting topics** take priority (recency = relevance)
2. **Recurring themes** across multiple meetings indicate deep priorities
3. **BANT scores** from AI analysis indicate buying readiness
4. **GTM relevance scores** indicate alignment with Wayground's value props
5. When multiple value props align, lead with the one matching the target persona
